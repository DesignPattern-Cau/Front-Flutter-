MyApp : 처음에 실행되는 함수
MainPage : 처음에 노출되는 페이지-도서검색,회원정보로 연결
UserInfo : 사용자 정보
UserInfoList : 사용자 정보에 담기는 내용을 입력
UserInfoCard : 사용자 정보에 내용을 넣는 틀
SearchPage : 책 검색 목록 페이지
InfoPage : 책 정보를 누르면 책의 정보와 관련 블로그 글이 나타나는 페이지
BlogPage : 블로그 글을 누르면 나타나는 페이지
BlogPostList : 관련 블로그 글의 내용을 입력
BlogPostCard : 관련 블로그 글의 틀
BookList : 책 정보의 내용을 입력
BookCard : 책 정보 틀
